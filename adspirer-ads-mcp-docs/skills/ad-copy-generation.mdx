# Ad Copy Generation

The Ad Copy Generation skill creates platform-optimized ad copy that aligns with your brand strategy.

## What It Does

1. **Reads your strategy** — Pulls brand voice, positioning, and USPs from `STRATEGY.md`
2. **Analyzes competitors** — References competitive landscape for differentiation
3. **Generates variants** — Creates multiple headline and description options
4. **Follows platform rules** — Respects character limits and best practices per platform
5. **Suggests improvements** — Recommends A/B test variants

## Example Usage

```
Write ad copy for my Google Search campaign targeting
"AI marketing automation" keywords
```

The skill generates responsive search ad headlines (up to 15) and descriptions (up to 4), optimized for quality score and click-through rate.
