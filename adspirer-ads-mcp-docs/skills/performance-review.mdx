# Performance Review

The Performance Review skill provides comprehensive campaign analysis with actionable recommendations.

## What It Analyzes

- **Key metrics** — ROAS, CPA, CTR, conversion rate, impression share
- **Trend analysis** — Period-over-period performance changes
- **Benchmark comparison** — Your metrics vs. industry benchmarks
- **Anomaly detection** — Unexpected performance changes with explanations
- **Budget efficiency** — Spend allocation and wasted budget identification

## Example Usage

```
Review my Google Ads performance for the last 30 days
and compare to the previous period
```

The skill pulls data from all connected platforms and provides a structured analysis with specific optimization actions.
