# Setup & Onboarding

The Setup skill initializes your Adspirer workspace and creates your brand strategy document.

## What It Does

1. **Connects accounts** — Links your Google Ads, Meta, LinkedIn, and TikTok accounts
2. **Pulls campaign data** — Imports existing campaigns and performance history
3. **Creates STRATEGY.md** — Builds your persistent brand strategy document
4. **Configures workspace** — Sets up your working environment for ongoing use

## Example Usage

In Claude Code:
```
/adspirer:setup
```

In other clients:
```
Set up my Adspirer workspace and connect my ad accounts
```
