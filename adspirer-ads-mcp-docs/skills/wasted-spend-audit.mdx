# Wasted Spend Audit

The Wasted Spend Audit skill identifies budget waste and provides specific actions to recover spend.

## What It Checks

- **Search term waste** — Irrelevant search terms consuming budget (Google Ads)
- **Underperforming campaigns** — Campaigns with poor ROAS or high CPA
- **Audience overlap** — Competing ad sets targeting the same users (Meta)
- **Creative fatigue** — Ad creatives with declining performance
- **Budget misallocation** — Spend not aligned with best-performing campaigns

## Example Usage

```
Run a wasted spend audit across all my ad platforms
```

The skill analyzes spend across all connected platforms and provides a prioritized list of actions with estimated savings.
