## Common Issues

<AccordionGroup>
  <Accordion title="OAuth authentication fails">
    Clear your browser cookies for `adspirer.com` and try reconnecting. If using Claude Code, run `/mcp` to check server status and re-authenticate.
  </Accordion>
  <Accordion title="Tools not appearing in my AI client">
    Ensure your AI client supports MCP. Check that the server URL is exactly `https://mcp.adspirer.com/mcp`. Restart your AI client after adding the server.
  </Accordion>
  <Accordion title="Tool call limit reached">
    Check your usage at [adspirer.ai](https://adspirer.ai). Consider upgrading your plan for more tool calls per month.
  </Accordion>
  <Accordion title="Ad platform not connecting">
    Make sure you have admin access to the ad account you're trying to connect. Some platforms require specific roles (e.g., Google Ads Manager account access).
  </Accordion>
</AccordionGroup>
