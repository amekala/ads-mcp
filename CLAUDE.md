# Adspirer — Performance Marketing Plugin

Manage ad campaigns across Google Ads, Meta Ads, LinkedIn Ads, and TikTok Ads via natural language.

## Getting Started

1. Run `/adspirer:setup` to connect your ad accounts and create your brand workspace
2. Authenticate when prompted via the browser
3. The setup wizard will scan your folder for brand docs and pull live campaign data

## Available Commands

| Command | What it does |
|---------|-------------|
| `/adspirer:setup` | Connect ad accounts and bootstrap your brand workspace |
| `/adspirer:performance-review` | Cross-platform performance analysis with KPI comparison |
| `/adspirer:wasted-spend` | Find and fix wasted ad spend across all platforms |
| `/adspirer:write-ad-copy` | Generate brand-voice-compliant ad copy |
| `/adspirer:refresh-brand-context` | Re-scan docs and update workspace with latest data |

## Safety

- All new campaigns are created in **PAUSED** status
- No existing campaigns will be deleted or have budgets modified without approval
- All actions require your explicit approval before execution
